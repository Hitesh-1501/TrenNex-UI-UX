TrenNex is a modern e-commerce mobile app UI/UX case study designed in Figma.

This project focuses on building clean user flows, simple interactions, and a smooth shopping experience.

---

## Project Goals
- Create a beginner-friendly e-commerce UI
- Design complete shopping flows
- Practice real-world mobile app UX
- Prepare for Android development

---

## Tools Used
- Figma
- Material Design guidelines
- Custom color system & typography
